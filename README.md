# GreenCare+ 🏥

A sustainable digital hospital management system built with **Next.js, Node.js, Express, and MongoDB** — designed to streamline hospital workflows while promoting eco-friendly healthcare practices.

## 🌟 Features

- **Patient Management** — Register, track, and manage patient records in real time
- **Doctor Assignment** — Assign and manage doctors across departments efficiently
- **Drug Tracking** — Monitor drug inventory and usage across the hospital
- **QR-Based Registration** — Paperless patient onboarding using QR codes
- **Analytics Dashboard** — Data-driven insights into hospital operations and resource utilization
- **Sustainability Metrics** — Track and reduce paper usage and carbon footprint
- **Secure Authentication** — Login and registration with protected routes

## 🛠️ Tech Stack

| Layer | Technology |
|-------|-----------|
| Frontend | Next.js, Tailwind CSS |
| Backend | Node.js, Express.js |
| Database | MongoDB |
| Language | TypeScript |

## 🚀 Getting Started

### Prerequisites
- Node.js v18+
- MongoDB (local or Atlas)

### Installation

```bash
# Clone the repository
git clone https://github.com/Spandana-n0204/GreenCare-plus.git

# Navigate to project directory
cd GreenCare-plus

# Install dependencies
npm install

# Create a .env.local file and add your MongoDB URI
MONGODB_URI=your_mongodb_connection_string

# Run the development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.
